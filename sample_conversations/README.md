# Conversation Examples

This folder contains examples of conversations that illustrate concepts discussed in the paper.
Please note that, for convenience, the examples were extracted from the original JSON conversation files and may not contain all fields.
All conversations come from real interactions with MAINLE, and similar interactions can be generated by executing the code in the [src](https://github.com/paulobruno/ecml-pkdd-2025/src/) folder.

## Visualizer

We provide a JSON visualizer to help you better read the conversations.
You can visualize the JSON history of conversations in a custom visualizer presented in two forms:
1. a web interface that you can access at [https://pb-rho.vercel.app/](https://pb-rho.vercel.app/); and
2. a standalone HTML file `conversation_visualizer.html`.
To use the standalone version, open the HTML file in your browser and load the conversation history you want to visualize.
