# Evaluation Data

This folder contains the data collected during the evaluation of MAINLE.

## User Study Data

The file `user_study.csv` contains the data collected during the user study.
No information that could identify the participants was collected, nor is it available.

## Critic Agent Data

The file `critic_agent_best.csv` contains the data collected from Critic agents when asked to provide the best explanation.
The file `critic_agent_rating.csv` contains the data collected from Critic agents when asked to rate the explanations provided by MAINLE.

## Sample Form

The file `sample_form.md` presents a sample form in markdown format containing the same fields as the form used in the evaluation. The original form was created using Google Forms, and the response URL was sent to each participant.
